CS1: Software Resources

For this class, we will be using the Python programming language. The Python home page is at http://www.python.org. We will be using Python version 2.7.10; that version is available here. Python documentation for this version can be found here. In particular, we recommend you look at the library (module) documentation, which is located here.

Python is installed on the CMS cluster; but if you want to install your own version, assignment 1 will walk you through the process. (You should definitely install Python on your computer; for one thing, Wing IDE may not be installed on the CMS cluster, so you would have to use a different setup.) If you have a computer running Linux or Mac OS X, chances are Python is already installed, though perhaps not the version we will be using. Having the correct version is important, because some of the Python libraries we'll be using may not work properly with other versions. In particular, using a later version of Python is not recommended.

The development environment we will be using to write our Python programs is called "Wing 101". It is available for free here. If you decide at some point that you really love this IDE and want the best version available, you can get a commercial version which has many additional features at the main WingWare site: http://www.wingware.com/. The free version will be adequate for our needs in this course.
